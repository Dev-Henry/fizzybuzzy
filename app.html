FIZZY BUZZY: A Coding Project

When it counts, you Fizzy Buzzy!
A JavaScript coding exercise using a standard loop (or and a boolean with a switch case statement or a ternary operator).
Write a program that prints the numbers from 1 to 100 and for multiples of '3' display "Fizz" instead of the of the number.
For the multiples of '5' display "Buzz". If a multiple of both, then display "FizzBuzz"

Instructions

Enter a fizzy and a buzzy value. Press the button and magic will happen.
